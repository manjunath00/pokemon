# Pokemon

A web application to view pokemons details
 
## Installation

``` 
git clone https://github.com/manjunath00/pokemon
cd pokemon

# create .env file & add this variable
DATABASE=your_database

# install and run via npm or yarn
npm i && npm run dev 
```
